(inner) find(table, search, orderByopt, descendingopt, columnopt)

Find rows which match a column.
Parameters:
Name Type Attributes Default Description
table The table to search.
search A Javascript object containing the keys: lhs (column name), op (comparison operator) and rhs (comparison value). If operator is "and" or "or", "lhs" and "rhs" are nested objects.
orderBy <optional>
 An optional column to sort by.
descending <optional>
 Whether to sort in descending (true) or ascending (false, null, undefined) order (default: ascending).
column <optional>
 * Whether to select just one or a few columns, or all of them (*). Default: *
